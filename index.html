🕐 Time Systems Converter
ℹ️ Instructions — Time Systems
  • Input: UTC date/time string (ISO 8601, e.g. 2025-06-21T12:00:00Z), or type now for current time, or JD:2451545.0 for Julian Date input
  • Output: Julian Date (JD), Modified Julian Date (MJD), Unix timestamp (ms), TAI (JD), TT (JD), GPS Week + TOW, leap seconds
Input

🌍 Earth Coordinate Transformations
ℹ️ Instructions — Earth Coordinates
  • Geodetic ↔ ECEF: Input lat (°, −90..90), lon (°, −180..180), alt (m). Output: X, Y, Z in meters (ECEF WGS-84)
  • ECEF ↔ ENU: Input reference lat/lon + ECEF delta. Output: East/North/Up in meters
  • ENU ↔ Az/El/Range: Input E, N, U in meters. Output: azimuth (°), elevation (°), range (m)
  • ECEF ↔ ECI: Input X, Y, Z + UTC time. Uses simplified GMST rotation
  • Observer → Target: Input observer and target lat/lon/alt. Output: Az/El/Range and above-horizon check
  • Solar Geometry: Input lat/lon + UTC. Output: sub-solar point, sunlit status, local solar time

🛸 Space & Orbital Mechanics
ℹ️ Instructions — Space/Orbital
  • Orbital Parameters: Input semi-major axis (m) and body. Output: period, circular/escape velocity
  • COE ↔ State: Input a (m), e, i (°), RAAN (°), ω (°), ν (°). Output: position/velocity vectors
  • Anomaly Conversions: Input eccentricity + mean anomaly. Output: eccentric and true anomaly
  • Hohmann Transfer: Input r₁, r₂ (m). Output: ΔV₁, ΔV₂, transfer time
  • Plane Change: Input velocity (m/s) + Δi (°). Output: ΔV

📡 Tracked Object Tools
⚠ Tracked-Object Pipeline — These tools parse TLE/OMM data for Earth-orbiting tracked objects. Propagation uses simplified two-body (Keplerian) math — not SGP4. Results are approximate and should not be used for operational tracking.
For generic orbital mechanics (COE, Hohmann, vis-viva), use the Space/Orbital tab.
ℹ️ Instructions — Tracked Object Tools
  • Parse TLE: Input: two-line element set (line 1 and line 2). Output: orbital elements, epoch, period
  • Parse OMM: Input: OMM JSON record (CelesTrak GP format). Output: same orbital elements
  • Propagate: Input: TLE/OMM + optional UTC time. Output: lat/lon/alt, ECI position (simplified 2-body, not SGP4)
  • Stream: Fetch live TLE/OMM data from CelesTrak. Select a source, click Fetch, then click a result to load it
